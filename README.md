# Home Loan Recommendation Chatbot

This is a simple AI-based chatbot that helps users get home loan recommendations based on their financial details.

## Features
- Chatbot interaction  
- Loan eligibility prediction  
- EMI calculation  
- Basic recommendations  

## Technologies Used
- Python  
- Machine Learning  

## How to Run
1. Install requirements:
   pip install -r requirements.txt  

2. Run the project:
   python home_loan_chatbot.py  

## Note
.env file is not included for security reasons.
